# LancerFi Documentation

Welcome to the LancerFi documentation! This guide will help you understand, set up, and use the LancerFi platform.

## What is LancerFi?

LancerFi is a decentralized freelancing platform built on the Solana blockchain. It enables secure escrow payments, allowing clients and freelancers to work together with trustless transactions powered by blockchain technology.
