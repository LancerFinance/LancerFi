# Prerequisites

Before setting up LancerFi, ensure you have the following installed:

## Required Software

### Node.js
- **Version**: Node.js 18 or higher
- **Installation**: Download from [nodejs.org](https://nodejs.org/)
- **Verification**: Run `node --version` in your terminal

### npm
- Comes with Node.js
- **Verification**: Run `npm --version` in your terminal

### Git
- **Installation**: Download from [git-scm.com](https://git-scm.com/)
- **Verification**: Run `git --version` in your terminal

### Phantom Wallet
- **Browser Extension**: Install from [phantom.app](https://phantom.app/)
- Required for wallet connection and testing

## Required Accounts

### Supabase Account
- Sign up at [supabase.com](https://supabase.com/)
- Create a new project
- Get your project URL and API keys

### Vercel Account (for deployment)
- Sign up at [vercel.com](https://vercel.com/)
- Install Vercel CLI: `npm i -g vercel`

## Optional but Recommended

### Code Editor
- **VS Code**: [code.visualstudio.com](https://code.visualstudio.com/)
- **Recommended Extensions**:
  - ESLint
  - Prettier
  - TypeScript

### Solana CLI (for advanced development)
- Install from [docs.solana.com](https://docs.solana.com/cli/install-solana-cli-tools)
