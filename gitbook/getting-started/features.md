# Features

LancerFi offers a comprehensive set of features designed for Web3 freelancing.

## Payment Features

### Multiple Payment Methods
- **SOL**: Pay with Solana's native token
- **USDC**: Stablecoin payments for price stability
- **X402**: X402 payment protocol integration

### Escrow System
- Funds are locked in blockchain escrow until project completion
- Automatic payment release upon approval
- No need to trust intermediaries

### Low Fees
- Only 1% platform fee
- No hidden charges
- Transparent fee structure

## Platform Features

### Project Management
- Create and post projects
- Browse available freelancers
- Submit and review proposals
- Track project progress

### Freelancer Profiles
- Create detailed profiles
- Showcase portfolio and skills
- Receive reviews and ratings
- Build reputation on-chain

### Messaging System
- Direct communication between clients and freelancers
- Real-time messaging
- Support ticket system

### Work Submission
- Submit work files and links
- Security scanning for uploaded files
- Review and approval workflow

## Security Features

### Wallet Authentication
- Connect with Phantom wallet
- Secure signature-based authentication
- No password required

### File Security
- Automatic file scanning
- NSFW content detection
- Suspicious file detection

### Rate Limiting
- Protection against spam
- Project creation limits
- Message rate limits

## Admin Features

### User Management
- User restriction system
- IP ban functionality
- Wallet ban capabilities
- Mute functionality

### Support System
- Admin support messages
- Rate limit management
- System status monitoring
