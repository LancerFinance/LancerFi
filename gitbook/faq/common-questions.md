# Common Questions

Frequently asked questions about LancerFi.

## General

### What is LancerFi?
LancerFi is a decentralized freelancing platform built on Solana blockchain. It enables secure escrow payments between clients and freelancers using blockchain technology.

### How is LancerFi different from other freelancing platforms?
- **Trustless**: No need to trust a central authority
- **Low Fees**: Only 1% platform fee
- **Fast**: Built on Solana for near-instant transactions
- **Transparent**: All transactions are on-chain
- **Web3 Native**: Built for the decentralized web

### What blockchain does LancerFi use?
LancerFi is built on the Solana blockchain.

## Payments

### What payment methods are supported?
- SOL (Solana native token)
- USDC (USD Coin stablecoin)
- X402 payment protocol

### How does the escrow system work?
Funds are locked in a blockchain escrow wallet when a project is funded. The funds are automatically released to the freelancer when the client approves the work.

### What is the platform fee?
The platform fee is 1% of the project value, automatically deducted from the escrow funding.

### When do I get paid?
Payment is automatically released when the client approves your submitted work.

## Wallet

### What wallet do I need?
You need a Phantom wallet (or compatible Solana wallet) to use LancerFi.

### Do I need to switch networks?
- **Development**: Use Solana Devnet
- **Production**: Use Solana Mainnet

Make sure your wallet is on the correct network.

### Is my wallet safe?
Yes. LancerFi never has access to your private keys. You only sign transactions, and funds are held in the platform escrow wallet.

## Projects

### How do I post a project?
1. Connect your wallet
2. Click "Post Project"
3. Fill in project details
4. Fund the escrow
5. Wait for proposals

### How do I find work as a freelancer?
1. Create a freelancer profile
2. Browse available projects
3. Submit proposals
4. Get selected and complete work

### Can I cancel a project?
Project cancellation depends on the project status. Contact support if you need to cancel a funded project.
